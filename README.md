🏨 LuxuryStay – Hotel Management System

LuxuryStay is a full-stack Hotel Management System built using the MERN Stack (MongoDB, Express.js, React, Node.js).
It allows hotels to manage rooms, bookings, users, and reservations through a modern dashboard with role-based access.

✨ Features
🔐 JWT Authentication
👤 Role-Based Access Control (Admin, Manager, Staff, Guest)
🏨 Room Management
📅 Booking & Reservation System
💳 Payment Simulation
📊 Admin Analytics Dashboard
📜 System Logs
⚙️ Hotel Settings Management


📈 Future Improvements
Stripe payment integration
Image uploads (Cloudinary)
Email notifications
Booking calendar view
